## 4. What shipped this session

### A. Terminal profiles + Claude (skip permissions) — `feature/terminal-profiles-skip-perms`
- **System shells as tabs:** `src/main/shell-profiles.ts` detects CMD / PowerShell
  / pwsh / Git Bash / WSL (Windows) and login shell + bash/zsh/fish/sh (POSIX).
  IPC `TERMINAL_LIST_SHELLS` / `TERMINAL_LAUNCH_SHELL`; a "Terminals" group in the
  `+` profile picker; shells render via `EmbeddedTerminal` in the `'other'`
  resource bucket.
- **Claude (skip permissions ⚠) picker entry:** per-launch
  `--dangerously-skip-permissions`, threaded `TerminalTab.skipPermissions` →
  `TerminalPanel` → `terminal.spawn` → `PtyManager` (OR-combined with the global
  Settings toggle; Claude PTYs only). Tab shows ⚠; persists.
- **`PtyRegistry.restart()`** now remembers per-pane launch params + category so a
  restart respawns the SAME thing (fixed: skip-perms/model command was dropped on
  restart). Red-team `docs/security-reviews/SECURITY_REVIEW_TERMINAL_PROFILES.md`.

### B. Catalyst Brain — `feature/obsidian-brain` (P1–P5 + extras)
"Brain" = Obsidian-compatible `.md` folder Catalyst owns directly. **We do NOT
ship/bundle the Obsidian binary (ToS).** "vault" stays reserved for the existing
compact-controller JSON sync — the Brain is never called a "vault" in code/UI.

| Module (`src/main/`) | Role |
|---|---|
| `brain-service.ts` | P1 — scoped `.md`/YAML/wikilink read+write; path guards; diff-before-write; optimistic-concurrency (SHA-256 hash); atomic tmp+rename; canvas/base read (`listSpecial`/`readCanvas`/`readBase`) |
| `brain-writer.ts` | P2 — canonical `BrainEntry` schema + bus → confined `_catalyst/<source>/<id>.md` (idempotent); used to mirror LMM/snippets/cost |
| `brain-index.ts` | P3 — RAG: chunk + embed via Ollama `/api/embeddings` (default `nomic-embed-text`), vectors in `<userData>/brain-index.json`, incremental rebuild, cosine search. Realizes BACKLOG #4. Gated on Ollama |
| `brain-graph.ts` | wikilink backlinks graph (cached; basename resolution) |
| `brain-rest-auth.ts` | P4 — Local REST API key (safeStorage) **+ live-vault client** (list/get/search/append/put/delete; Node http/https; self-signed cert accepted **loopback-only**) |
| renderer `components/brain/BrainPanel.tsx` | the 🧠 sidebar panel: folder picker, searchable list, editor + diff preview + conflict-safe save, backlinks, semantic search, mirror, canvas/base, copy/send-to-tab context, REST live ops |
| `obsidian-plugin/` | P5 — first-party Obsidian plugin scaffold (MIT typings + esbuild). Build separately: `cd obsidian-plugin && npm i && npm run build` |

IPC: all `brain:*` channels in `src/shared/ipc-channels.ts`; preload `brain`
namespace; types `Brain*` in `src/shared/types.ts`; window typings in
`src/declarations.d.ts`. Wiring in `src/main/index.ts` `setupBrain()`. Sidebar
entry + `'brain'` panel case in `App.tsx` + `Sidebar.tsx`; `'brain'` in
`session-service.ts` `VALID_PANEL_IDS` + `SessionPanelId`.
Red-teams: `SECURITY_REVIEW_BRAIN_{P1,P2_P4,P5}.md`.

## 5. Architecture quick-map (for PTY / launch work)

- **Claude tab:** `TerminalPanel` → `terminal.spawn(paneId, cwd, skipPermissions)`
  → `PtyRegistry.spawn` → `PtyManager.spawn`. No `opts.command` ⇒ Claude; flag
  injected from `cli-flags.json` (global) OR per-launch `skipPermissions`.
- **Model / shell tab:** `models.launch` / `terminal.launchShell` pre-spawn a PTY
  (`model:…` / `shell:…` paneId), rendered by `EmbeddedTerminal` (attaches, never
  spawns). Model PTYs never get the skip-perms flag.
- **Brain:** all writes go through `BrainService.writeNote` (path-scoped, atomic).
  RAG/REST never write into the user's vault index location (vectors live in
  `userData`).

## 8. Conventions / gotchas

- **Node 22 only** (`engines.node ">=22.0.0 <24.0.0"`); Node 24 silently breaks
  electron-builder packaging.
- **`npm run dist` needs Windows Developer Mode ON** (winCodeSign symlinks).
- **Multi-agent:** more than one AI agent works this repo (+ multiple machines).
  Read `AGENT_COORDINATION.md`, **claim a row before editing**, push the claim
  early. Cross-agent notes go in that file's Decisions log.
- **LMM + red-team discipline:** apply the Lincoln Manifold Method on non-trivial
  work; red-team + remediate after each phase, write to
  `docs/security-reviews/`; fix Criticals+Highs same commit, defer Mediums with a
  note.
- **Prepackaging guardrail:** never bundle/auto-install the Obsidian binary
  (proprietary ToS). The Brain is native; real Obsidian is optional BYO.
- End commit messages with the project's Co-Authored-By trailer.
